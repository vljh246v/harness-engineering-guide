# Tech Debt Tracker

즉시 수정하지 않고 보류한 기술 부채를 추적합니다.
해결 시 해당 행을 삭제하고 커밋 메시지에 `refs: TD-XXX`를 포함합니다.

## 활성 부채

| ID | 설명 | 심각도 | 등록일 | 관련 파일 |
|----|------|--------|--------|-----------|
| (프로젝트에 맞게 추가) | | | | |

## 심각도 기준

| 등급 | 의미 | 대응 |
|------|------|------|
| HIGH | 장애 위험 또는 보안 취약점 | 1주 내 해결 |
| MEDIUM | 개발 생산성 저하 | 1개월 내 해결 |
| LOW | 코드 품질 개선 | 분기 내 해결 |

## 해결 방법

1. 해당 행의 작업을 `task-start.sh`로 시작
2. 수정 후 `task-finish.sh`로 병합
3. 이 파일에서 해당 행 삭제
4. 커밋 메시지에 `refs: TD-XXX` 포함
